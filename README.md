# 404 Error Page

A visually stunning 404 error page with animated robot, neon effects, and interactive elements.

## Features

- 🤖 Animated robot character
- ✨ Neon glow effects on the 404 text
- 🎨 Dynamic background with floating particles
- 🖱️ Interactive click explosions
- 🎭 3D tilt effect on mouse movement
- 🚀 Fun easter eggs (click the robot!)

## Technologies

- HTML5
- CSS3 (animations, gradients, transforms)
- Vanilla JavaScript

## Usage

Simply open `404.html` in a web browser or integrate it into your website as your custom 404 error page.

## Demo Effects

- **Glitch Animation**: The 404 text has a glitch effect
- **Floating Container**: Smooth floating animation
- **Particle Background**: Randomly generated colored particles
- **Click Explosions**: Click anywhere to create particle explosions
- **Robot Interactions**: Click the robot for surprise effects

## Browser Support

Works in all modern browsers that support CSS3 animations and ES6 JavaScript.
